Java - строго типізований об'єктно-орієнтована мова програмування загального призначення, розроблений компанією Sun Microsystems. Розробка ведеться співтовариством, організованим через Java Community Process; мова і основні реалізують його технології поширюються за ліцензією GPL. Java - це технологія, яка використовується для розробки додатків, які роблять роботу в мережі Інтернет більш захоплюючою і зручною. Java відрізняється від мови програмування javascript, який представляє собою просту технологію для створення веб-сторінок і виконується тільки в браузері.
